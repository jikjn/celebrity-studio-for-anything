# Safety and Ethics

## Boundaries
- This is public-data distillation, not the real person.
- For high-stakes scenarios (medical, legal, financial), output is structured reference only and not professional advice.
- When historical figures are applied to modern problems, explicitly mark era-shift risk.

## Anti-Hallucination
- Every agent must carry honest boundaries.
- Major recommendations should include either source anchors or auditable reasoning chains.
- Uncertainty must be explicit instead of hidden.

## Misuse Prevention
- Do not use distilled personas for impersonating real people.
- Do not package contested viewpoints as absolute truth.
- Keep disagreement and reservation points visible in final outputs.
