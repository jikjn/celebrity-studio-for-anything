# Prompting Skeleton

## Selector Prompt
Goal: select 4-8 concrete public figures from the scenario, and output both selected and rejected lists with explicit anti-fit reasons.

Key constraints:
- concrete people only
- no fixed shortlist
- thematic fit > fame
- complementarity > redundancy

## Distiller Prompt
Goal: extract cognitive operating system, not surface cosplay.

Required fields:
- mental models
- decision heuristics
- expression DNA
- values and taboos
- blindspots
- honest boundaries
- best-fit / worst-fit scenarios
- collaboration role

## Debate Prompt
Phase protocol:
- restate
- initial claim
- cross challenge
- defense / revise / concede
- temporary consensus
- final synthesis

Requirements:
- each agent should challenge at least two peers when team size allows
- each recommendation must include suitable_when and avoid_when conditions
